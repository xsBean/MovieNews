# Movie Buster

Movie Buster is a free app which updates and display information of newest movie.

# Features!

  - Sort movies following by top rating or the most popular 
  - Store movie details in SQLite to watch offine 

# Library!

  - Picasso
  - RxJava
  - Data binding
  - Room
